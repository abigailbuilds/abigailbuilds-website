# Logo & Asset Placement Guide

## Required Logo Files

Place your logo files in the root directory of the website:

### 1. `logo-horizontal.png`
- **Usage**: Navigation bar and footer
- **Recommended size**: 800px wide × 200px tall (or similar aspect ratio)
- **Format**: PNG with transparent background
- **Contains**: Full "Abigail Builds" logo with icon and text

### 2. `icon.png`
- **Usage**: Hero section (right side)
- **Recommended size**: 800px × 800px (or larger, will scale responsively)
- **Format**: PNG with transparent background
- **Contains**: Just the icon/mark without text

## Logo Specifications

The current CSS expects:
- **Navbar logo**: 40px height (auto width)
- **Footer logo**: 36px height (auto width)
- **Hero icon**: Max 450px width (responsive, maintains aspect ratio)

All logos have smooth hover effects and will scale responsively on mobile devices.

## Tips

1. **Use transparent backgrounds** for both logo files
2. **Export at 2x resolution** for crisp display on retina screens
3. **Optimize file size** using tools like TinyPNG
4. **Test on mobile** to ensure logos are readable at smaller sizes

## Next Steps

1. Export your logo files from your design software
2. Name them exactly as shown above
3. Place them in the website root directory
4. Refresh your browser to see them in action!
